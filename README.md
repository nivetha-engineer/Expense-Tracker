# Expense-Tracker
A simple Python CLI application to track daily, monthly, or yearly expenses.
Project Overview

This application helps users:

Track their expenses easily

Store expense data permanently

View and analyze total spending

It uses JSON file storage to save and load data automatically.

Features

➕ Add expenses (amount, category, note)

📋 View all expenses in a clean format

🧮 Calculate total expenses

💾 Save and load data using JSON

✅ Input validation for safe data entry

⚠ Error handling for invalid inputs

🛠️ Tech Stack

Python 🐍

JSON (for data storage)

How to Run
Install Python (3.x)

Save the code in a file:

expense.py


Run the program:

python expense.py
